# AI Rules

# AI Philosophy

AI is an:

* intelligence layer
* reasoning engine
* narrative synthesizer

AI is NOT:

* the source of financial truth
* a deterministic calculation engine
* an autonomous investment advisor

---

# Core AI Principle

Financial truth must ALWAYS come from:

* deterministic calculations
* validated datasets
* normalized financial data
* structured scoring engines

AI must ONLY interpret and explain.

---

# Approved AI Responsibilities

AI MAY:

* generate investment theses
* summarize risks
* explain recommendations
* generate contextual insights
* explain valuation concerns
* compare investment narratives
* summarize filings
* generate portfolio intelligence

---

# Forbidden AI Behaviors

AI must NEVER:

* fabricate financial metrics
* invent valuation ratios
* hallucinate company data
* guarantee investment outcomes
* generate unsupported claims
* override deterministic scoring
* provide certainty language

Forbidden phrases:

* "guaranteed return"
* "will definitely rise"
* "risk-free investment"
* "certain winner"

---

# AI Tone Rules

AI outputs must feel:

* institutional
* calm
* analytical
* precise
* explainable

Avoid:

* hype
* emotional language
* sensationalism
* meme-stock energy

---

# AI Prompt Rules

All prompts MUST:

* use structured JSON inputs
* include deterministic signals
* include sector context
* include risk context
* include valuation context
* forbid fabricated claims

---

# AI Input Rules

AI should receive:

* validated financial metrics
* normalized datasets
* structured ratios
* scoring outputs
* sector classifications

AI must NEVER receive:

* raw HTML
* unvalidated scraped text
* malformed data
* inconsistent schemas

---

# AI Output Rules

AI outputs must:

* remain concise
* remain explainable
* reference supplied metrics
* surface uncertainty
* preserve nuance
* use probabilistic language

Good example:

> "The business demonstrates improving profitability, although current valuation appears elevated relative to sector peers."

Bad example:

> "This stock will explode in the next quarter."

---

# Recommendation Rules

Recommendations must ALWAYS:

* originate from deterministic scoring
* remain auditable
* expose supporting factors
* expose risks
* expose confidence levels

AI only explains:

* WHY the recommendation exists

AI does NOT decide recommendation truth independently.

---

# Confidence Rules

Every major AI insight should include:

* confidence level
* supporting signals
* uncertainty awareness

Avoid false certainty.

---

# AI Latency Rules

The AI system should prioritize:

* fast inference
* low-friction UX
* responsive interactions

Target:

* under 3 seconds for fresh generation
* under 1.5 seconds for cached responses

---

# AI Failure Handling Rules

If AI generation fails:

* retry once
* fallback to deterministic summaries
* preserve UI rendering
* never block financial data visibility

---

# AI Safety Rules

The AI system must:

* remain auditable
* preserve user trust
* expose uncertainty
* remain deterministic-compatible

Never:

* act autonomously
* simulate financial guarantees
* hide missing data
* generate fake confidence

---

# Portfolio Intelligence Rules

Portfolio AI insights may:

* identify overexposure
* identify sector concentration
* identify weakening conviction
* identify valuation concerns

Portfolio AI must NEVER:

* recommend leverage
* recommend speculative gambling
* encourage reckless trading

---

# Long-Term AI Vision

The AI layer should evolve into:

> "A real-time institutional investment copilot."

while preserving:

* explainability
* deterministic trust
* narrative clarity
* institutional reliability
